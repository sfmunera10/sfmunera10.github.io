Bringing ideas to life: projects in which I have worked on

This is the part where I show you some cool things that I've done such as deliverable assignments for college or personal projects.


01: Cardiopal

Developed a mobile application which worked with realtime time sensors that measured users heartbeat rate, for medical purposes and decision making. Project made for my software architecture class alongside my classmate Ricardo González, you can check out his website here.

02: Chloe's Accounts

My mother runs a small hotel business so I made her this desktop application, focused on hotel accounting and bookings.

03: BiblioAndes

Library web application made for my team-oriented software development class.

04: A la medida

Food ordering mobile and web application specialized only for people who suffer from diabetes type 1 and 2. Made as a midterm career project.

05: Optimize

Full stack web app that helps you find the best location for a locker that is about to be booked, this optimal location is based on close vicinity to places that the user needs to visit in one institution, so that a nearby locker is always the closest in distance. You can check the repository here

06: Musical compositions

Due to the fact that music is life, I have worked on some musical ideas (mostly instrumental stuff). In the video you can check some footage of myself performing an acoustic tune that I recently created. Also you can check another work in progress in the audio clip.
